HOW IT WORKS
------------
1. User requests business creation: "Create a SaaS for task management"
2. GenesisMetaAgent builds detailed requirements
3. ProductGenerator creates prompt for Claude Sonnet 4
4. Claude generates complete application (15-25 files)
5. ProductValidator scans for security/quality issues
6. Files deployed to Vercel with setup instructions
7. Live application URL returned to user
