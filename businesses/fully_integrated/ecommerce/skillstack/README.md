# SkillStack

AI-powered marketplace that automatically creates personalized learning paths by combining micro-courses from multiple creators, then delivers them as subscription boxes with physical materials and digital content. Each box contains curated tools, templates, and exclusive creator content tailored to the learner's skill progression and industry needs.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
