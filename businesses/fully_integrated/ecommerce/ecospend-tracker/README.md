# EcoSpend Tracker

AI-powered personal finance app that automatically tracks your carbon footprint and suggests sustainable purchasing alternatives while helping you save money. Users get rewarded with cashback and discounts from eco-friendly brands for making greener spending choices.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
