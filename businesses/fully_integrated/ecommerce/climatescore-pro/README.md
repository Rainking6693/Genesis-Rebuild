# ClimateScore Pro

AI-powered sustainability compliance platform that automatically generates ESG reports and carbon footprint assessments for small businesses. Combines climate tech with micro-SaaS automation to help companies meet regulatory requirements and win eco-conscious customers.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
