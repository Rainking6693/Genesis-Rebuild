# MeetingMind AI

AI-powered meeting intelligence platform that automatically generates personalized action items, follow-up emails, and project briefs from recorded meetings. Transforms chaotic meeting discussions into structured, actionable deliverables that actually get completed.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
