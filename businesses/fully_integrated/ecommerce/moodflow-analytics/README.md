# MoodFlow Analytics

AI-powered workplace mental health analytics platform that helps small businesses track team wellness patterns and automatically suggests personalized interventions. Combines anonymous mood tracking with actionable business insights to reduce turnover and boost productivity.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
