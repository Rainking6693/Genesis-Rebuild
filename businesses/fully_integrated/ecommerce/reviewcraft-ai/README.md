# ReviewCraft AI

AI-powered platform that helps small e-commerce businesses automatically generate authentic, personalized product review responses and customer follow-up campaigns to boost retention and social proof. Combines AI productivity tools with community-driven templates and no-code automation workflows.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
