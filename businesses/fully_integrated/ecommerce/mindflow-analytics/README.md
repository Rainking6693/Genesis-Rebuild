# MindFlow Analytics

AI-powered mental wellness dashboard that helps small businesses reduce employee burnout by analyzing team communication patterns and automatically suggesting personalized wellness interventions. Combines workplace productivity monitoring with proactive mental health support to boost retention and performance.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
