# CreatorCRM Pro

AI-powered CRM specifically designed for content creators to automatically segment their audience, personalize outreach, and convert followers into paying customers across all platforms. Unlike generic CRMs, it understands creator-specific metrics like engagement rates, content performance, and fan loyalty to optimize monetization strategies.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
