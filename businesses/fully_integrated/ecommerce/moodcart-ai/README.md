# MoodCart AI

AI-powered personalized product recommendations that adapt to users' current emotional state and mental wellness goals. Combines mood tracking with curated product subscriptions to deliver items that genuinely improve users' daily mental health and productivity.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
