# CarbonCred

AI-powered platform that automatically tracks small businesses' carbon footprint through financial transactions and sells them verified carbon credits while providing sustainability scoring for competitive advantage. Businesses get instant ESG compliance, cost savings through green recommendations, and marketing credibility with minimal setup effort.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
