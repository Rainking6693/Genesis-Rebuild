# MoodSync Pro

AI-powered team wellness platform that analyzes remote team mood patterns through daily micro-surveys and automatically suggests personalized productivity tools, break schedules, and team-building activities. Combines mental health monitoring with AI productivity optimization to boost both employee wellbeing and business performance metrics.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
