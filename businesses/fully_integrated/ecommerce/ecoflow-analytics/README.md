# EcoFlow Analytics

AI-powered sustainability tracking platform that automatically calculates and optimizes carbon footprints for small e-commerce businesses while generating customer-facing sustainability reports. Transforms climate compliance from a cost center into a competitive advantage and revenue driver through automated green marketing content.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
