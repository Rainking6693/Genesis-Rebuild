# Review Rocket

AI-powered platform that automatically generates personalized review request campaigns for e-commerce businesses, turning one-time buyers into loyal brand advocates through intelligent timing and messaging. Combines automation tools with creator economy mechanics by gamifying the review process and rewarding customers with exclusive perks.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
