# EcoTrack Pro

AI-powered sustainability analytics platform that automatically tracks and certifies small businesses' environmental impact across their entire supply chain. Generates compliance reports, carbon offset recommendations, and sustainability badges that boost customer trust and sales conversion rates.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
