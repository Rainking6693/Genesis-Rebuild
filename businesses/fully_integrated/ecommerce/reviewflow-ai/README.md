# ReviewFlow AI

AI-powered platform that automatically generates personalized customer review response templates and sentiment-driven upsell campaigns for e-commerce stores. Transforms negative reviews into retention opportunities and positive reviews into viral marketing moments through intelligent automation.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
