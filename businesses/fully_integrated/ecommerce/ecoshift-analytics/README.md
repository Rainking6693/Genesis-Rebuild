# EcoShift Analytics

AI-powered sustainability assessment platform that helps small e-commerce businesses instantly calculate, optimize, and market their environmental impact while turning green practices into competitive advantages. We transform complex carbon footprint data into actionable insights and automated marketing content that drives both sustainability and sales.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
