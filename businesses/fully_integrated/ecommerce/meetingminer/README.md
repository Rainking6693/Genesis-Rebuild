# MeetingMiner

AI-powered SaaS that transforms recorded team meetings into actionable business intelligence dashboards, tracking team sentiment, productivity patterns, and decision-making bottlenecks. Small businesses get executive-level insights from their everyday conversations without expensive consultants.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
