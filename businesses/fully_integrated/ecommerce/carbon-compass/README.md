# Carbon Compass

AI-powered carbon footprint tracking and offset marketplace that helps remote teams automatically calculate their work-from-home emissions and purchase verified offsets. Creates competitive team challenges and integrates with popular work tools to gamify sustainability while building company culture.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
