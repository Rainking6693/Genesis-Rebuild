# SkillStack AI

AI-powered platform that creates personalized learning paths for small business teams, automatically curating micro-courses and tracking skill development to boost productivity. Combines educational content subscription with AI-driven workforce analytics to help SMBs identify and close skill gaps efficiently.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
