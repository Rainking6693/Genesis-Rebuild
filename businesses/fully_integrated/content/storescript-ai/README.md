# StoreScript AI

AI-powered platform that automatically generates personalized video sales scripts and email sequences for e-commerce store owners based on their product data and customer behavior. Transform product catalogs into high-converting video content and automated email campaigns within minutes using no-code workflows.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
