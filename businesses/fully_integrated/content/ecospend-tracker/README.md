# EcoSpend Tracker

AI-powered platform that tracks your spending's environmental impact and provides personalized recommendations to reduce your carbon footprint while saving money. Combines real-time carbon scoring with financial wellness coaching through gamified challenges and community leaderboards.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
