# RetentionLab

AI-powered churn prediction and customer retention content generator that creates personalized email sequences, in-app messages, and loyalty program content for e-commerce brands. Combines predictive analytics with automated content creation to help online stores retain customers before they churn.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
