# Course Compass

AI-powered platform that analyzes small business employees' skill gaps and automatically curates personalized learning paths from existing online courses across multiple platforms. Transforms scattered corporate training into a unified, measurable system that tracks ROI and predicts performance improvements.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
