# WellnessFlow AI

AI-powered platform that automatically generates personalized employee wellness content calendars for small businesses, complete with daily micro-learning modules, team challenges, and ROI tracking. Combines health & wellness with remote work productivity by delivering bite-sized, actionable wellness content that reduces sick days and boosts team engagement.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
