# MoodSync Pro

AI-powered platform that analyzes team sentiment from Slack/Teams messages and automatically generates personalized mental health content, productivity insights, and manager coaching recommendations. Combines real-time workplace mood analytics with actionable wellness content delivery to prevent burnout before it happens.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
