# MoodMetrics

AI-powered micro-SaaS that generates personalized mental wellness content and actionable insights for small business teams based on real-time mood tracking and productivity patterns. Helps HR managers reduce burnout and boost team performance through data-driven wellness interventions.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
