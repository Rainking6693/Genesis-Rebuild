# MindFlow Studio

AI-powered platform that transforms workplace stress and productivity challenges into personalized micro-learning content and wellness workflows for small business teams. Combines mental health insights with productivity optimization through bite-sized, actionable content delivered at optimal moments throughout the workday.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
