# ClimateCase Studio

AI-powered platform that automatically generates compelling sustainability case studies and ROI reports for B2B companies to showcase their climate initiatives to stakeholders and customers. Combines real business data with climate impact metrics to create professional, shareable content that drives both ESG compliance and marketing value.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
