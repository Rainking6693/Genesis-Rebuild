# EcoSell Academy

A micro-learning platform that teaches sustainable e-commerce practices through bite-sized video courses and AI-generated implementation checklists. We turn eco-conscious business owners into profitable sustainability experts while building a community of certified 'Green Commerce' professionals.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
