# MoodFlow Analytics

AI-powered platform that tracks team emotional wellness through anonymous micro-surveys and generates actionable mental health insights for managers to improve productivity and retention. Combines mental health monitoring with business intelligence to help small businesses prevent burnout before it impacts their bottom line.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
