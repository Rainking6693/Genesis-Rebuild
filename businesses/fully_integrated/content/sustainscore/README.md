# SustainScore

AI-powered platform that generates personalized sustainability scorecards and actionable improvement plans for small businesses, helping them track, improve, and showcase their environmental impact to customers. Creates automated content marketing materials and compliance reports while building a community marketplace for sustainable business practices.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
