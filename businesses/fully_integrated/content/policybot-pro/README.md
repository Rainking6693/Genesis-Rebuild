# PolicyBot Pro

AI-powered platform that automatically generates, updates, and manages custom business policies (privacy, terms, HR, compliance) for small businesses. Eliminates expensive lawyer consultations while ensuring legal compliance through intelligent document creation and regulatory change monitoring.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
