# CarbonCopy

AI-powered platform that automatically generates personalized sustainability reports and carbon footprint dashboards for small e-commerce businesses, helping them attract eco-conscious customers and comply with emerging green regulations. Creates shareable, branded content that drives customer trust and viral marketing through transparent environmental impact storytelling.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
