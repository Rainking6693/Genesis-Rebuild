# Carbon Credits

AI-powered platform that automatically generates personalized sustainability reports and carbon offset subscription boxes for small businesses. Combines climate compliance tracking with curated eco-friendly products delivered monthly to help companies meet ESG goals while engaging employees.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
