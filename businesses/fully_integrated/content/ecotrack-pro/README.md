# EcoTrack Pro

AI-powered carbon footprint tracking and sustainability reporting platform that helps small businesses automatically monitor their environmental impact and generate compliance reports for customers and investors. Creates a community marketplace where businesses can discover, implement, and share verified eco-friendly practices while earning sustainability scores that boost their market credibility.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
