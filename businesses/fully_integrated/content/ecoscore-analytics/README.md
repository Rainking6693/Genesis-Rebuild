# EcoScore Analytics

AI-powered sustainability scoring platform that generates automated ESG compliance reports and carbon footprint dashboards for small businesses. Transforms complex environmental regulations into actionable insights with white-label content templates for consultants and agencies.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
