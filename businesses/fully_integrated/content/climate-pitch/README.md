# Climate Pitch

AI-powered platform that generates customized sustainability content and carbon impact reports for small businesses to attract eco-conscious customers. Combines climate data with marketing automation to help businesses showcase their environmental efforts through compelling, data-driven content.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
