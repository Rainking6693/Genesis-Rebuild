# EcoMetrics Pro

An AI-powered sustainability tracking and content automation platform that helps e-commerce businesses generate compliant ESG reports, carbon footprint content, and eco-marketing materials while building customer trust through transparency. It combines real-time environmental data with automated content generation to turn sustainability compliance into a competitive marketing advantage.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
