# MoodMetrics Pro

AI-powered employee wellness dashboard that correlates team mood patterns with productivity metrics, helping small businesses predict burnout and optimize work schedules. Combines anonymous mood tracking with performance analytics to provide actionable insights for managers while protecting employee privacy.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
