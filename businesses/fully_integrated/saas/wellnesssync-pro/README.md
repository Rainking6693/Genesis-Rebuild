# WellnessSync Pro

AI-powered employee wellness platform that automatically integrates with existing business tools to track team stress levels, suggest micro-interventions, and prevent burnout before it happens. Uses behavioral data patterns from Slack, calendar apps, and work metrics to provide personalized wellness recommendations and real-time mental health insights for managers.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
