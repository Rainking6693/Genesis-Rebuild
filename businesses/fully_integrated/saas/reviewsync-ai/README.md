# ReviewSync AI

AI-powered review management platform that automatically monitors, responds to, and analyzes customer reviews across all platforms while generating personalized response templates that match your brand voice. It combines community-driven best practices with no-code automation to help small businesses maintain their online reputation without hiring dedicated staff.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
