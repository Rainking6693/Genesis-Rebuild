# EcoSpend Tracker

AI-powered expense tracking that automatically calculates and offsets your carbon footprint while helping you save money through eco-friendly spending recommendations. Gamifies sustainable living by turning every purchase into climate action with real-time impact visualization and community challenges.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
