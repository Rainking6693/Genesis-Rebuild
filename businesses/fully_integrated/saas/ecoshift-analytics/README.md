# EcoShift Analytics

AI-powered sustainability analytics platform that helps e-commerce businesses automatically track, optimize, and market their environmental impact to eco-conscious consumers. Transform sustainability compliance from a cost center into a competitive advantage and revenue driver through automated carbon tracking, supplier scoring, and customer-facing impact dashboards.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
