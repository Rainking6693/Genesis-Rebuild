# EcoFlow Analytics

AI-powered sustainability analytics platform that automatically tracks and optimizes small businesses' carbon footprint while identifying cost-saving opportunities through energy efficiency and waste reduction. Combines real-time expense monitoring with climate impact scoring to help SMBs meet ESG requirements and reduce operational costs simultaneously.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
