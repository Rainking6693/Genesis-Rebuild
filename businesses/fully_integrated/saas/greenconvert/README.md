# GreenConvert

AI-powered sustainability analytics platform that automatically calculates and optimizes the carbon footprint of e-commerce businesses while turning eco-friendly practices into customer acquisition tools. Transforms environmental compliance from a cost center into a revenue-generating marketing advantage through automated sustainability badges, carbon offset upsells, and green loyalty programs.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
