# RetentionBot AI

AI-powered customer retention automation that analyzes e-commerce customer behavior patterns and automatically triggers personalized re-engagement campaigns before customers churn. Combines predictive AI with sustainable business practices by helping companies retain existing customers rather than constantly acquiring new ones.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
