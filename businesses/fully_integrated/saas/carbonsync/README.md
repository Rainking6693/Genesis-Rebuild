# CarbonSync

AI-powered carbon footprint tracker that automatically calculates and offsets business expenses in real-time by connecting to accounting software and suggesting sustainable alternatives. Transforms every business purchase into an opportunity for climate action while reducing costs through green vendor recommendations.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
