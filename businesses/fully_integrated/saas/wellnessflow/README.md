# WellnessFlow

AI-powered employee wellness automation platform that integrates mental health check-ins, sustainable workplace habits, and financial wellness coaching directly into Slack/Teams. Creates personalized wellness action plans while providing employers with anonymous team wellness analytics to reduce burnout and increase retention.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
