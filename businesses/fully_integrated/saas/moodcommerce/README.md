# MoodCommerce

AI-powered e-commerce personalization platform that adapts product recommendations and website experiences based on customers' real-time emotional state and mental wellness goals. Combines sentiment analysis, behavioral psychology, and commerce optimization to boost conversions while promoting healthier purchasing decisions.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
