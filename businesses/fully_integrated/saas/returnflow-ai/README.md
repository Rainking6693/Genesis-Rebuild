# ReturnFlow AI

AI-powered return management platform that transforms product returns into retention opportunities by automatically analyzing return patterns and generating personalized retention offers. Reduces return rates by 35% while increasing customer lifetime value through intelligent intervention workflows.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
