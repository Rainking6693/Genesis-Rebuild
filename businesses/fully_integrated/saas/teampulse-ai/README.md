# TeamPulse AI

AI-powered employee wellness and productivity platform that automatically detects team burnout signals through work patterns and communication sentiment analysis. Provides actionable wellness interventions and productivity optimizations while maintaining complete privacy and anonymity.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
