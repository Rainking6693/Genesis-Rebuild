# CreatorSync

AI-powered subscription box curation platform that enables content creators to automatically generate, manage, and fulfill personalized product boxes for their audiences. Combines creator economy monetization with sustainable e-commerce by matching creators with eco-friendly suppliers and handling the entire logistics chain.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
