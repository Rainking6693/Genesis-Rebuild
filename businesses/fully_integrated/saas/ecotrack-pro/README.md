# EcoTrack Pro

AI-powered carbon footprint tracking and compliance platform that automatically calculates, reports, and offsets business emissions while generating ESG reports for small businesses. Combines real-time expense scanning with climate action recommendations to help companies meet sustainability goals and attract eco-conscious customers.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
