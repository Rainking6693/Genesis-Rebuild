# MoodSync Pro

AI-powered team wellness platform that analyzes anonymous mood patterns to predict burnout, optimize workloads, and automatically suggest mental health interventions before productivity crashes. Combines real-time sentiment analysis from Slack/Teams with automated workflow adjustments and personalized wellness recommendations.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
