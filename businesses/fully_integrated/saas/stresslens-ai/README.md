# StressLens AI

AI-powered workplace stress monitoring platform that analyzes team communication patterns (Slack, email, calendar) to predict burnout risks and automatically suggests personalized wellness interventions. Combines mental health insights with automation to help small businesses retain talent and boost productivity before stress becomes costly turnover.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
