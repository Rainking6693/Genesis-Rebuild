# MoodBoard Analytics

AI-powered employee wellness platform that analyzes team mood patterns through daily micro-check-ins and provides actionable productivity insights to managers. Combines mental health monitoring with AI-driven productivity optimization to reduce burnout and increase team performance.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
