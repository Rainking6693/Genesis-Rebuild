# ReturnSense AI

AI-powered return prediction and prevention platform that helps e-commerce businesses reduce return rates by 30-50% through smart product recommendations and pre-purchase risk scoring. Combines climate impact tracking with cost savings to appeal to sustainability-conscious merchants and their customers.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
