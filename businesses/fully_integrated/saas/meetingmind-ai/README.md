# MeetingMind AI

AI-powered meeting intelligence platform that automatically transforms remote team meetings into actionable business insights, tracking team wellness indicators and generating personalized productivity recommendations. Goes beyond basic transcription to provide emotional sentiment analysis, decision tracking, and automated follow-up task assignment with wellness-focused workload balancing.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
