# Creator Compass

AI-powered platform that helps small businesses discover, vet, and manage micro-influencer partnerships through automated matching and campaign tracking. Combines creator economy insights with subscription-based relationship management to turn influencer marketing into a predictable revenue channel.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
