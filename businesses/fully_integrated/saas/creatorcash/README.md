# CreatorCash

AI-powered financial dashboard that automatically tracks income from multiple creator platforms (YouTube, TikTok, Substack, etc.) and provides real-time tax optimization suggestions. Combines creator economy tools with personal finance automation to solve the nightmare of multi-platform income tracking for content creators.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
