# ReviewFlow AI

AI-powered customer review management platform that automatically generates personalized, authentic responses to reviews across all platforms while detecting fake reviews and providing sentiment-driven business insights. Combines GPT-4 with brand voice training to help small businesses maintain their online reputation without hiring expensive marketing agencies.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
