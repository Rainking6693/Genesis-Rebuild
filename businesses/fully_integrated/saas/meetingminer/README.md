# MeetingMiner

AI-powered meeting intelligence platform that automatically extracts actionable tasks, decisions, and follow-ups from video calls and generates smart project workflows. Transforms chaotic meeting outputs into organized, trackable business outcomes with zero manual effort.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
