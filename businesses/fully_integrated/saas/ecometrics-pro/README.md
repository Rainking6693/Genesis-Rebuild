# EcoMetrics Pro

AI-powered sustainability analytics platform that automatically tracks, calculates, and optimizes carbon footprints for small e-commerce businesses while generating customer-facing sustainability reports. Helps online retailers turn environmental compliance into a competitive advantage and marketing tool.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
