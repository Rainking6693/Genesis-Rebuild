# ReturnFlow

AI-powered return management platform that automatically processes e-commerce returns, generates smart restocking decisions, and transforms return data into actionable business insights. Reduces return processing time by 80% while turning returns from a cost center into a profit optimization engine.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
