# CarbonTrail

AI-powered carbon footprint tracking and offset marketplace specifically designed for e-commerce businesses to automatically calculate, display, and neutralize the environmental impact of every customer purchase. Transforms climate guilt into competitive advantage by enabling transparent sustainability and automated carbon-neutral shopping experiences.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
