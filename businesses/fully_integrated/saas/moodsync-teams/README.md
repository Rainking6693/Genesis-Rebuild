# MoodSync Teams

AI-powered emotional intelligence platform that analyzes team communication patterns across Slack/Teams to predict burnout, suggest wellbeing interventions, and optimize team dynamics for remote/hybrid workforces. Combines mental health monitoring with productivity insights to help managers proactively support their teams before issues escalate.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
